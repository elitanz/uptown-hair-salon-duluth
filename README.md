# Uptown Hair Salon — Duluth, MN

A custom one-page website for **Uptown Hair Salon**, a full-service salon on
Piedmont Avenue in Duluth, Minnesota, serving the neighborhood since 1995.

**Live site:** https://elitanz.github.io/uptown-hair-salon-duluth/

- 📍 2820 Piedmont Ave B, Duluth, MN 55811
- ☎️ (218) 727-5256
- ⭐ 4.8 across 41 Google reviews
- 💇 Cuts & styling · color · extensions · nail care · waxing

Built with hand-written HTML/CSS/JS (no framework). Features a still-photo
image-dominant hero, scroll-reveal motion, count-up stats, a real verbatim
Google-review slider, a Google Maps embed, a FormSubmit.co appointment-request
form, click-to-text, and a sticky mobile call bar. All photography is licensed
stock (Pexels) pending the salon's own photos.
